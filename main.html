Leading Projects RecursiveCipher
RecursiveCipher is an advanced encryption algorithm that transforms plaintext into ciphertext through a multi-stage process. It operates across 10 versions, each mapping letters to 100 unique symbols. The encryption begins by randomly selecting a version and encoding each letter into 100 randomized symbols, which are sent in blocks to complicate interception by attackers, and complicating the methods of MITM attackers. Upon receipt, RecursiveCipher uses recursion to decrypt the message, ensuring secure and reliable communication.
PGD
PGD is a project that consists of three sophisticated codes leveraging TensorFlow and Mediapipe. The first code tracks hand gestures in real-time, accurately recognizing various movements using a trained model. The second code implements finger position tracking, effectively turning the index finger into a mouse cursor, allowing for precise movement and clicking actions based on finger location. The third code adjusts the system volume by measuring the distance between the thumb and index finger, providing a dynamic and intuitive volume control mechanism. These codes collectively enable seamless and efficient computer interaction through hand gestures.
